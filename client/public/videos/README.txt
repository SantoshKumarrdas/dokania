Place your video file(s) here to serve them from the site root.

nFor example, to use video.mp4 in the VideoSection component, add:

n  client/public/videos/video.mp4

nThen in your React code you can reference it as:

n  <VideoSection videoUrl="/videos/video.mp4" />

nSupported formats: mp4 (H.264) is recommended for broad browser support.
